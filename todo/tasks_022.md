# Task 022: Implement Task List Panel

## Objective
Create task list view with table widget.

## Dependencies
- Task 021 (TUI framework)

## Implementation
- Table widget with columns: ID, Title, Status, Type, Created
- Row selection with up/down arrows
- Status color coding (green=completed, red=failed, yellow=running)
- Auto-refresh from gRPC list calls
- Pagination support

## Complexity: Moderate | Effort: 5-6 hours
